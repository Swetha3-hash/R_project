# Bellabeat Case Study — Google Data Analytics Capstone

## 📌 Overview
This project is the capstone case study for the **Google Data Analytics Professional Certificate**. It analyses smart device fitness data to uncover usage trends and generate actionable marketing recommendations for **Bellabeat**, a wellness technology company focused on women's health.

---

## 🎯 Business Task
Identify trends in smart device usage and apply insights to inform Bellabeat's marketing strategy for one of their products.

---

## 🛠️ Tools Used
- **R** — data cleaning, analysis, and visualisation
- **ggplot2** — data visualisation
- **tidyverse** — data manipulation

---

## 📊 Key Steps
1. **Ask** — Defined the business problem and key questions
2. **Prepare** — Loaded and reviewed the FitBit fitness tracker dataset
3. **Process** — Cleaned data, handled missing values, formatted dates
4. **Analyse** — Identified activity trends, sleep patterns, and device usage habits
5. **Share** — Created visualisations to communicate findings clearly
6. **Act** — Delivered marketing recommendations based on data insights

---

## 💡 Key Findings
- Most users are lightly active — opportunity to encourage more engagement
- Sleep tracking is underutilised — potential for targeted marketing campaigns
- Usage drops on weekends — suggests habit-based engagement strategies
